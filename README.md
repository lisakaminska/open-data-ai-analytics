# Аналіз відкритих даних: Особи, які переховуються від органів влади (Версія B)

**Мета проєкту:** Провести статистичний та візуальний аналіз даних МВС щодо осіб у розшуку для виявлення закономірностей за категоріями та регіонами.

**Джерело даних:** [Інформація про осіб, які переховуються від органів влади](https://data.gov.ua/dataset/59ecf2ab-47a1-4fae-a63c-fe5007d68130)

## Питання та гіпотези для аналізу:
1. **Розподіл за статтями:** Які статті обвинувачення зустрічаються найчастіше серед осіб у розшуку
2. **Географічний чинник:** У яких регіонах (органах розшуку) зафіксована найбільша кількість осіб, що переховуються
3. **Демографічний аналіз:** Чи існують помітні закономірності у розподілі осіб за статтю або віковими категоріями

# Аналіз відкритих даних: Особи, які переховуються від органів влади (Версія A+B)

**Мета проєкту:** Провести технічний аналіз якості даних та комп'ютерне дослідження (Computer Vision) характеристик зображень осіб у розшуку для оцінки їх придатності до автоматизованого розпізнавання.

**Джерело даних:** [Інформація про осіб, які переховуються від органів влади](https://data.gov.ua/dataset/59ecf2ab-47a1-4fae-a63c-fe5007d68130)

## Питання та гіпотези для аналізу:
1. **Цілісність даних:** Перевірка наявності дублікатів ID та повноти заповнення технічних полів.
2. **Аналіз зображень (CV):** Дослідження роздільної здатності фото та їх форматування (JPEG/PNG).
3. **Придатність для AI:** Визначення відсотка фотографій, які мають достатню чіткість та розмір для моделей розпізнавання облич.
